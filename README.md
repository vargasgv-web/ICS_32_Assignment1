# ICS_32_Assignment1
A Simple File Explorer. 
Will be able to Create, Delete, and Read Files in stated directory. 
Takes input as one long string (Ex: R /Users/victor/Projects/a1_test_2026-04-24-11-14-56/Zzz/read.dsu) 
and returns the users desired command (if it is an actual command). 
Everytime a command is completed/failed, the user is allowed to input another command. 
User can Type Q to quit program.
